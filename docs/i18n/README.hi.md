## इंस्टॉलेशन

```bash
npm install entroly-wasm && npx entroly-wasm
# या
pip install entroly && entroly go
```

बस इतना ही। यह आपकी IDE को ऑटो-डिटेक्ट करता है, Claude/Cursor/Copilot/Codex/MiniMax में जुड़ जाता है, और कंप्रेशन शुरू कर देता है।

## स्मार्ट होने की लागत $0 क्यों — तीन स्तंभ

**1. टोकन इकोनॉमी** — `ValueTracker` लाइफटाइम बचत `S(t)` मापता है। इवोल्यूशन बजट सख्ती से सीमित: `C_spent(t) ≤ τ · S(t) (τ = 5%)`

**2. स्ट्रक्चरल इंडक्शन ($0)** — किसी भी टोकन को छूने से पहले, डिटरमिनिस्टिक सिंथेसाइज़र आपके कोड का AST, डिपेंडेंसी एज और एंट्रॉपी ग्रेडिएंट पढ़ता है। कोई LLM नहीं। कोई एम्बेडिंग नहीं। कोई क्लाउड नहीं।

**3. ड्रीमिंग लूप** — 60 सेकंड से अधिक आइडल होने पर, सिस्टम सिंथेटिक क्वेरीज़ जेनरेट करता है, स्कोरिंग वेट्स को पर्टर्ब करता है, और बेंचमार्क्स पर सेल्फ-प्ले करता है।
