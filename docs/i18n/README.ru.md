## Установка

```bash
npm install entroly-wasm && npx entroly-wasm
# или
pip install entroly && entroly go
```

Всё. Система определяет вашу IDE, подключается к Claude/Cursor/Copilot/Codex/MiniMax и начинает сжатие.

---

## Почему обучение стоит $0 — Три столпа

**1. Токен-экономика** — `ValueTracker` измеряет накопленную экономию `S(t)`. Бюджет эволюции строго ограничен: `C_spent(t) ≤ τ · S(t) (τ = 5%)`

**2. Структурная индукция ($0)** — До использования любого токена детерминированный синтезатор читает AST, рёбра зависимостей и градиент энтропии вашего кода и создаёт рабочий инструмент. Без LLM. Без эмбеддингов. Без облака.

**3. Цикл сновидений** — При простое > 60 сек система генерирует синтетические запросы, возмущает весовые коэффициенты и играет сама с собой на бенчмарках. Строгие улучшения сохраняются; регрессии отбрасываются.
